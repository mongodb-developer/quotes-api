Welcome to the Quotes APIs!

The available resources are:

  • /random: returns a random quote.
    Try it!
  • /search?q=<query>: searches for quotes that matech a certain string (with fuzzy matching).
    Try it!

Other fun things we support:

  • /quote.html: pretty and colorful view of your favorite quotes.
    Try it!

    Runs this query: [ { $match: { colors: { $ne: null } } }, { $sample: { size: 1 } } ]
